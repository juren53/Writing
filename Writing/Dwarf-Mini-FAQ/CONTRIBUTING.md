# Contributing to the Dwarf Mini FAQ

Thank you for helping improve this community resource!

---

## How to Suggest a Question or Correction

### Option 1 — Open an Issue (easiest)

1. Go to the [Issues tab](../../issues) and click **New Issue**.
2. Choose one of the issue templates:
   - **New Question** — suggest a question and answer to add
   - **Correction** — report an error or outdated information
   - **General Feedback** — anything else
3. Fill in the details and submit.

You don't need to write polished text — a rough description of the problem or question is enough. A maintainer will follow up.

### Option 2 — Submit a Pull Request

If you're comfortable with GitHub and Markdown:

1. Fork this repository.
2. Edit `README.md` (or other files as appropriate).
3. Follow the style guidelines below.
4. Submit a Pull Request with a brief description of your change.

---

## Style Guidelines

- **Q: / A: format** — Each entry starts with `**Q: ...**` on its own line, followed by the answer as a plain paragraph (no `A:` label needed — the answer follows directly).
- **Keep answers concise.** Aim for clarity over completeness. Link to official resources rather than reproducing lengthy documentation.
- **Flag version-specific information.** If something depends on a specific firmware or app version, say so explicitly. E.g., *"As of firmware v1.2..."*
- **Avoid speculation.** If you're not sure, say so or leave it out. Incorrect information is worse than a gap.
- **No affiliate links.** External links should go to official or well-established community resources only.

---

## What Makes a Good FAQ Entry?

A good FAQ entry answers a question that:
- Real users actually ask (not hypothetical edge cases)
- Is not clearly answered in the official DWARFLAB documentation
- Has an answer that is stable enough to be worth writing down

A good answer:
- Is accurate as of a recent firmware/app version
- Notes when information may be version-dependent
- Is written in plain language accessible to a beginner

---

## Scope

This FAQ covers the **Dwarf Mini** specifically. Questions about the Dwarf II or Dwarf 3 are out of scope unless the answer applies equally to the Dwarf Mini.
